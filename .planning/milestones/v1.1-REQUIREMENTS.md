# Requirements: EmoHub v1.1 UX Polish

**Defined:** 2026-02-12
**Core Value:** 用户能快速给表情包打标签并通过标签搜索找到想要的图片

## v1.1 Requirements

Requirements for v1.1 UX Polish milestone. All shipped.

### 外观与主题 (Theme)

- [x] **THEME-01**: 用户可以在 light/dark/system 三种主题间切换 — Phase 6
- [x] **THEME-02**: 用户选择的主题偏好在刷新后保持不变 — Phase 6
- [x] **THEME-03**: 选择 system 时自动跟随操作系统主题变化 — Phase 6

### 国际化 (i18n)

- [x] **I18N-01**: 用户可以在中文和英文之间切换界面语言 — Phase 7
- [x] **I18N-02**: 用户选择的语言偏好在刷新后保持不变 — Phase 7
- [x] **I18N-03**: 所有 UI 文本通过翻译键渲染，无硬编码字符串 — Phase 7

### 图片操作 (Image Actions)

- [x] **IMG-01**: 用户可以一键复制图片到系统剪贴板 — Phase 8
- [x] **IMG-02**: 用户复制时可以选择原图格式或转换为 GIF 格式 — Phase 8
- [x] **IMG-03**: 复制操作完成后用户收到成功/失败反馈 — Phase 8

### 视觉打磨 (Visual Polish)

- [x] **VISUAL-01**: 按钮、卡片、侧边栏等组件具有清晰的视觉层次区分 — Phase 8
- [x] **VISUAL-02**: 图片加载时显示骨架屏，空状态显示引导提示 — Phase 8
- [x] **VISUAL-03**: 交互元素具有平滑过渡动画（hover、modal、主题切换） — Phase 9
- [x] **VISUAL-04**: 动画尊重用户的 `prefers-reduced-motion` 系统设置 — Phase 9

### 设置 (Settings)

- [x] **SET-01**: 用户可以通过设置页面集中管理主题和语言偏好 — Phase 5
- [x] **SET-02**: 设置变更即时生效，无需刷新页面 — Phase 5
